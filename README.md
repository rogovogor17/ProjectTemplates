## About The Project

I'm sure each of you copied the main project files the next time you started your brilliant inventions. I want to create a Project template so amazing that it'll be the last one you ever need -- I think this is it.

Here's why:

- Your time should be focused on creating something amazing. A project that solves a problem and helps others
- You shouldn't be doing the same tasks over and over like creating a native from scratch
- You should implement DRY principles to the rest of your life.

Of course, no one template will serve all projects since your needs may be different. So I'll be adding more in the near future. You may also suggest changes by forking this repo and creating a pull request or opening an issue. Thanks to all the people have contributed to expanding this template!

## Getting Started

Most likely, you will only need one of the templates provided in this repository, and a complete copy of it, without information about the state of the repository itself, which makes `git clone` useless.

### Prerequisites

In order to copy a specific project template to your directory you will need [degit](https://github.com/Rich-Harris/degit):

```sh
npm install -g degit
```

### Installation

1. Choose a project template you want to copy.
2. Copy the repo

   ```sh
   #For instance, you chose Cpp project
   #In case when your destination directory is not empty, you may need `--force` flag

   degit --force rogovogor17/ProjectTemplates/Cpp
   ```

3. Don't forget to correct Project Name in root `CMakeLists.txt` in C like projects
